# Gestor de tareas

Aplicación de escritorio desarrollada con JavaFX para la interfaz gráfica y Spring Boot como backend, utilizando MySQL como base de datos.

Permite gestionar tareas de forma simple e intuitiva: crear, editar, eliminar y marcar tareas como completadas.

## Tecnologías utilizadas

- Java 17
- JavaFX
- Spring boot
- MySQL
- Maven
- JPA / Hibernate

  ## Características

- Crear nuevas tareas
- Editar tareas existentes
- Eliminar tareas
- Marcar el estado de la tarea
- Filtrar por estado (pendientes / completadas)
